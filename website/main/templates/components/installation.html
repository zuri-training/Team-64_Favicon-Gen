Installation

First, use the download button to download a zip file containing the files listed below. Place the files in the root directory of your website.

File sizes:
  • 16*16: Browser favicon
  • 32*32: Taskbar shortcut favicon
  • 96*96: Desktop shortcut favicon
  • 180*180: Apple touch favicon

File formats:
  • PNG, JPG, SVG, ICO

Also, the related embed codes will be included in the zip file. Copy and paste them into the head of your HTML. Examples are shown below.

<link rel="apple-touch-icon" sizes="180x180" href="/apple-touch-icon.png">
<link rel="icon" type="image/png" sizes="32x32" href="/favicon-32x32.png">
<link rel="icon" type="image/png" sizes="16x16" href="/favicon-16x16.png">
<link rel="icon" type="image/png" sizes="96x96" href="/favicon-96x96.png">
